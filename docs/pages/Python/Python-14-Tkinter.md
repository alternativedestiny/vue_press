# Tkinter

## 1. Tkinter 简介

1. Python 内置图形界面包

## 2. 使用

1. 创建界面

    ```python
    from tkinter import *

    master = Tk()

    # 窗口标题
    master.title("窗口标题")
    # 窗口大小 300px*150px 坐标起始位置：左 300px, 上 100px
    master.geometry("300x150+300+100")  # 窗口呈现大小和位置

    master.mainloop()
    ```
