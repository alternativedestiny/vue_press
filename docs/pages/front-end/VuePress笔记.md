# VuePress 笔记

## 1. 安装配置

1. 环境
   - node v12.14.1
   - npm 6.13.4

2. 安装 VuePress

    ```bash
    # 安装
    npm install -g vuepress  # 可以使用cnpm
    ```

## 2. 创建项目

1. 项目启动

    ```bash
    npm run dev
    ```

2. 推送到github

    ```bash
    npm run deploy
    ```
