# Vue3 学习笔记

## Vue-cli 配置

1. 安装[Node](https://nodejs.org/zh-cn/)
2. 配置cnpm

    ``` BASH
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    ```

3. 安装`Vue-cli`

    ``` BASH
    npm i -g @vue/cli
    ```

4. 创建vue项目

    ``` BASH
    vue create my_project
    ```

5. 启动项目

    ``` BASH
    cd my_project
    npm run serve
    ```

6. 启动UI管理界面

    ``` BASH
    vue ui
    ```
